4G Network Planning

This one is one part of the challenging project which help me to gain more knowledge regarding network planning especially radio frequency networks. Right before planning, the one that should be prepared is the link budget to calculate the parameters and how many sites will be needed for the coverage area. Next, preparing the initial setup likes types of network, project area, data clutter, setting LTE parameter, and so on. For this project, the frequency chosen is 1800 MHz for FDD 4G LTE. The important thing that must be set up is defining clutter losses that depend on the types of area. That will be better to define because there are types of areas in one planning location. After that, set up and calculate each of the sites and cells. For the final, check and visualized the signal level coverage of the planning.
